1. Var vs let
	var keyword allows you to reassign and redeclare variables
	let keyword doesn't allow you to redeclare variables 
	You can access a hoisted variable with var. Let creates block scope.
2. Var vs const
You can reassign and redeclare with var, but you can not redeclare or reassign using the const keyword.
You can access hoisted variable with car. Const creates a block scope.

3. Let vs const
You can reassign with let, but you can not redeclare or reassign using the const keyword.

4. What is hoisting?
	Hoisting is lifting the variable to the top of the scope they are declared in. It means that when using var, you can access the variable name and its undefined value before it is used later on.
